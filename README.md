# DSZI_Survival

Projekt symulatora agenta surwiwalowego w dzikim środowisku, przygotowywany na przedmiot *Sztuczna Inteligencja*.
Skład zespołu:
- Jonatan Spaczyński
- Mateusz Tylka
- Michał Czekański
- Marcin Kostrzewski

## Wymagania
```
Python
pygame
```
## Uruchomienie
```
$ python Run.py
```
## Konfiguracja
Plik z konfiguracją znajduje w ```data/config/mainConfig.json```.
